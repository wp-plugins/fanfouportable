== Description ==

A simple tool about [Fanfou](http://fanfou.com/),create new status when you publish a new post,and get the recent statuses from Fanfou.

1. Admin page

== Installation ==

1. Upload `fanfouportable` to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Change options,enter you fanfou ID and password.
